## El Juego

La verdad es que ahora mismo el juego no es gran cosa; es muy simple y no tiene muchos elementos pero tiene algunas cosas fundamentales que pueden servir de estructura para la creación de un videojuego.

En cualquier caso, creo que seguiré actualizándolo cuando tenga tiempo libre.

Dejo aquí el enlace por si hay alguien que quiera descargarlo para jugar o le sirve de ayuda para su propio proyecto. (se puede jugar en Windows y en Linux, pincha en el enlace para más información)

[godot_2dgame](https://github.com/albergv/godot_2dgame)


## Godot

El motor gráfico que estoy usando es Godot. De código abierto, en estos últimos años en mi opinión ha avanzado bastante.

Tiene una documentación amplia y fácil de entender y cada vez hay más personas usándolo (lo que proporciona una mayor comunidad para resolver dudas o aprender)
